O que é HTML, CSS e JavaScript?

HTML

HTML é a linguagem de marcação que estrutura o conteúdo da página.

<p>Olá mundo</p>

Quando usar: sempre, para estruturar conteúdo.

CSS

CSS define a apresentação visual dos elementos estruturados em HTML.

p { color: #2563EB; }

Quando usar: para layout, temas e responsividade.

JavaScript

JavaScript traz interatividade e lógica ao site no navegador.

document.querySelector('p').textContent = 'Olá';

Quando usar: para comportamentos, validações e manipulação DOM.
